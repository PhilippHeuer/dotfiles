# dotfiles

## installation

- clone this repository
- run `./setup.sh` to symblink the config files from this repository.
- add the init scripts to your `.bashrc`: `. $HOME/.config/bash/init.sh`

## authentication

- copilot: run `:copilot auth` and follow the instructions
